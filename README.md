# Homebrew Tap — Claude Usage Monitor

macOS menu bar app for monitoring Claude Code usage limits (session 5h / weekly 7d).

## Install

```bash
brew tap eunggg0/tap
brew install --cask claude-usage-monitor
```

## Requirements

- macOS 12.0 (Monterey) or later
- [Claude desktop app](https://claude.ai/download) installed and logged in

## Update

```bash
brew upgrade --cask claude-usage-monitor
```

## Uninstall

```bash
brew uninstall --cask claude-usage-monitor
```
